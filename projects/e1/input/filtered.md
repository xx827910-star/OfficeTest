# 中国科学技术大学本科毕业论文格式规范（可脚本化版本）

## 页面设置

- 纸张大小：A4

## 摘要

### 中文摘要

**标题样式：**
- 标题文字："摘 要"
- 字体：黑体
- 字号：三号
- 对齐：居中
- 行距：单倍行距
- 段前间距：24磅
- 段后间距：18磅

**内容样式：**
- 字体：中文用宋体，英文和数字用Times New Roman
- 字号：小四
- 行距：固定20磅
- 段前间距：0磅
- 段后间距：0磅

**关键词格式：**
- 标题："关键词："加粗
- 位置：另起一行，隔行排列于摘要下方
- 对齐：左顶格
- 悬挂缩进：4个字符
- 分隔符：用分号（；）隔开

### 英文摘要（ABSTRACT）

**标题样式：**
- 标题文字："ABSTRACT"
- 字体：Times New Roman
- 字号：三号
- 加粗：是
- 对齐：居中
- 行距：单倍行距
- 段前间距：24磅
- 段后间距：18磅

**内容样式：**
- 字体：Times New Roman（英文和数字）
- 字号：小四
- 行距：固定20磅
- 段前间距：0磅
- 段后间距：0磅

**关键词格式：**
- 标题："Key Words:"加粗
- 位置：另起一行
- 对齐：左顶格
- 悬挂缩进：6.5个字符
- 每个关键词首字母大写
- 分隔符：用分号（;）隔开

## 目录

- 标题文字："目 录"
- 字体：黑体
- 字号：三号
- 行距：单倍行距
- 段前间距：24磅

**目录项样式：**

**页码设置：**
- 行末右顶格

## 正文

### 一级标题（章）

**示例1格式：**
- 格式："第一章"、"第二章"等
- 字体：黑体
- 字号：三号
- 段前间距：根据示例默认设置
- 段后间距：根据示例默认设置

**示例2格式：**
- 格式："第1章"、"第2章"等
- 字体：黑体
- 字号：三号

**示例3格式：**
- 格式："1"、"2"等
- 字体：黑体
- 字号：三号

**注意：** 本规范采用示例1格式（第一章、第二章等）

### 二级标题（节）

**示例1格式：**
- 格式："第一节"、"第二节"等
- 字体：黑体
- 字号：小三

**示例2格式：**
- 格式："1.1"、"1.2"等
- 字体：黑体
- 字号：小三

**示例3格式：**
- 格式："1.1"、"1.2"等
- 字体：黑体
- 字号：小三

**注意：** 本规范采用示例1格式（第一节、第二节等）

### 三级标题（段）

**示例1格式：**
- 格式："一、"、"二、"等
- 字体：黑体
- 字号：四号

**示例2格式：**
- 格式："1.1.1"、"1.1.2"等
- 字体：黑体
- 字号：四号

**示例3格式：**
- 格式："1.1.1"、"1.1.2"等
- 字体：黑体
- 字号：四号

**注意：** 本规范采用示例1格式（一、二、等）

### 正文段落

**段落样式：**
- 字体：中文用宋体，英文和数字用Times New Roman
- 字号：小四
- 行距：固定20磅
- 首行缩进：2字符
- 段前间距：0磅
- 段后间距：0磅

**编号规则：**
- 第四级："1."、"2."等
- 第五级："（1）"、"（2）"等
- **注意：** 在标题中，数字序号和汉字序号不可混用

## 表格

**表格编号和标题：**
- 编号格式："表1"、"表2"等（全文统一编号）或按章编号
- 字体：宋体
- 字号：小四
- 位置：居表上方
- 对齐：居中

**表格内容：**
- 字体：宋体
- 字号：小四或五号（选定某种样式后，全文应统一）

**表格样式：**
- 采用三线表（顶线、栏目线、底线）

## 图片

**图片编号和标题：**
- 编号格式：全文统一编号或按章编号
- 字体：宋体
- 字号：小四
- 位置：居图下方
- 对齐：居中

## 公式

**公式格式：**
- 位置：单独占行
- 对齐：居中
- 行距：单倍行距

**公式编号：**
- 格式：用括号括起，如（1）、（2）等
- 位置：置于公式右边行末
- 字体：与段落文字格式相同
- 字号：与段落文字格式相同（小四）
- 行距：与段落文字格式相同（固定20磅）
- 编号与公式间：不加任何连线

**字体规范：**
- 变量：斜体
- 常量和特殊函数：正体
- 矢量：黑斜体
- 张量：黑正体

## 参考文献

**标题样式：**
- 标题文字："参 考 文 献"
- 字体：黑体
- 字号：三号
- 行距：单倍行距
- 段前间距：24磅
- 段后间距：18磅

**条目格式：**
- 编号方式：顺序编码制（1. 2. 3. ...）
- 字体：中文用宋体，英文用Times New Roman
- 字号：小四

**著录格式示例：**

专著：
```
作者.书名[M].出版地：出版社，出版年.
作者.书名[M].出版地：出版社，出版年:起止页码.
```

期刊：
```
作者.文章标题[J].刊名，年，卷（期）:起止页码.
```

学位论文：
```
作者.论文题名[博士学位论文].保存地：保存单位，年份.
```

## 过滤说明

本文档已从原始格式规范中过滤掉以下内容：

### 已删除的章节

- **封面** - 原因：包含作者信息、学号、导师等个人信息，属于非毕业论文文档本身的内容
- **学术诚信承诺书** - 原因：包含签名，无法自动生成
- **原创性声明和版权授权书相关内容** - 原因：包含签名相关内容，需要手工处理

### 已删除的细节要求

- **"摘要一般300～500个汉字"** - 原因：属于内容层面的要求，需要理解内容才能判断，无法脚本化
- **"英文摘要的篇幅参照中文摘要"** - 原因：属于内容层面的要求
- **"摘要应简要说明论文的研究目的、基本研究内容、研究方法或过程、结果和结论，突出论文的创新之处"** - 原因：属于内容质量要求，需要人工判断
- **"摘要应具有独立性和自明性"** - 原因：属于内容质量要求，需要人工判断
- **"摘要中不宜使用公式、图表，不引用文献"** - 原因：需要内容理解和人工判断
- **"关键词应选择论文主题特色的专业名词术语，突出、强调论文的重点和核心内容"** - 原因：需要内容理解，无法脚本化
- **"建议关键词数量不超过8个"** - 原因：属于内容层面的建议，非强制格式要求
- **"作者姓名、学号、专业、导师、完成时间"等个人信息** - 原因：属于非格式化的个人信息填写
- **装订顺序和打印相关要求** - 原因：与打印装订相关，非文档格式本身

### 已保留但需说明的内容

- **表格内容字号** - 原文提供"小四宋体或五号宋体"两种选择，本规范统一为小四宋体
- **标题编号格式** - 原文提供三种示例格式，本规范统一采用示例1格式（第一章、第一节、一、等中文格式）
- **公式字体规范** - 保留了变量、常量、矢量、张量的字体要求（正斜体、黑白体），这些可以通过MathType或公式编辑器的样式设置实现
- **表格、图片、公式的字体等信息** - 这些细项有自己的定义，优先级高于全文字体设置

### 格式统一说明

原文档提供了多种编号格式示例（例1、例2、例3），为便于脚本化处理，本规范统一采用：
- 章：第一章、第二章...
- 节：第一节、第二节...
- 段：一、二、三...
- 更细分级：1. 2. 3... 和 （1）（2）（3）...

此格式选择基于原文示例1，符合中文学术规范。
